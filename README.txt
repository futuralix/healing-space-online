THERAPY SERVICES WEBSITE
========================

This is a simple HTML/CSS/JavaScript website for a psychotherapy practice.

FILES INCLUDED:
- index.html (Home page)
- about.html (About Me page)
- services.html (Services page)
- contact.html (Contact page)
- styles.css (All styling)
- script.js (Interactive features)
- README.txt (This file)

HOW TO USE:
1. Copy all files to a folder on your USB stick
2. Open index.html in any web browser
3. Navigate between pages using the top navigation menu

FEATURES:
- Responsive design (works on mobile, tablet, and desktop)
- Mobile-friendly navigation menu
- Smooth scrolling
- Contact form with validation
- Modern, professional design
- All pages linked together

CUSTOMIZATION:
To customize the content:
1. Open any HTML file in a text editor (Notepad, TextEdit, etc.)
2. Find the text you want to change
3. Edit and save the file
4. Refresh your browser to see changes

To change colors:
1. Open styles.css in a text editor
2. Find the :root section at the top
3. Modify the color values
4. Save and refresh

BROWSER COMPATIBILITY:
Works in all modern browsers including:
- Chrome
- Firefox
- Safari
- Edge

NO INTERNET CONNECTION REQUIRED - All files run locally!
